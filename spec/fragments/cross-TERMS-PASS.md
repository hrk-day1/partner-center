## 목적

조각 명세 전반에서 **같은 개념이 같은 한국어/영문 키**로 읽히도록 교차 점검한 결과를 한 곳에 모은다.

## 표준 용어 (권장)

| 개념 | 권장 표기 | 비고 |
|------|-----------|------|
| 관리자 앱 좌측 첫 메뉴 | **홈** | 내비 라벨; 본문은 **홈 큐레이션**. |
| 방문자용 크리에이터 페이지 | **공개 홈** / **크리에이터 홈** | `prof-PR2`, `home-*` |
| 로그인 없이 열람 | **전체 공개** | 홈 행·포스트 권한 공통. |
| 유료 티어(상품 연동) | **OO 구독** (베이직·프리미엄·VIP) | 상품명은 `mem-M2` 매핑표 따름. |
| 홈 큐레이션 행에서의 짧은 라벨 | 예: **VIP 멤버십** | 포스트·상품의 **VIP 구독**과 **동일 티어**인지 `mem-M2`에 명시. |
| 큐레이션 묶음 | **큐레이션 그룹** | `home-H4`~`H6` |
| 포스트 노출 범위 | **가시성** / **열람 권한** | 홈은 가시성, 포스트 편집은 열람 권한 설정. |
| 저장 버튼(상단) | **변경사항 저장** | `shell-G3` |

## 상태·뱃지 (교차)

| 맥락 | 값 예시 | 조각 |
|------|---------|------|
| 포스트 | 임시저장 / 발행됨 / 예약됨 | `post-PM2` |
| 멤버십 상품 | 판매중 / 판매중단 / 준비중 | `mem-M1` |
| 멤버십 상세 | 정상 등 | `mem-M1-detail` |
| 홈 상단 | 홈페이지 공개 중 | `home-H1` |

## 일관성 점검 결과

- [x] **공개**: 전체 공개·그룹/행 공개·포스트 공개 설정이 혼동되지 않도록 각 조각에서 한글 **공개** 유지.
- [x] **구독 vs 멤버십**: 상품·결제 맥락은 **구독**(베이직 구독 …); 방문자 화면 짧은 라벨에 **멤버십**이 쓰이면 `mem-M2` 매핑으로 연결.
- [x] **VIP**: **VIP 멤버십**(홈) ↔ **VIP 구독**(포스트/상품) — **표기 통일은 제품 사전에서 확정**(잔여: [`cross-open-issues.md`](cross-open-issues.md) O-10 인접 항목).
- [x] **미리보기 / 변경사항 저장**: `shell-G3` 판정표를 단일 기준으로 함.
