# H-3 — 고정 포스트 (PIN)

<!-- CP-HOME: 2026-03-31 부록 관측 반영 -->

## UI

- 카드: **고정 포스트**
- **PIN** 배지.
- 썸네일·제목 일부(부록 예: **봄 시즌 …**).
- 가시성: **전체 공개** 및 행 단위 **공개** 표시(눈 아이콘·녹색 강조).
- **해제** (고정 취소) 텍스트 버튼.
- **연필(편집)** 아이콘 — 포스트 메타 편집 또는 교체 진입으로 해석 가능 → 동작은 `TBD`/제품 정의.
- 하단 안내 문구(부록): 고정 포스트가 **크리에이터 홈 최상단**에 노출된다는 설명.

## 동작

| 액션 | 시스템 반응 (초안) |
|------|---------------------|
| 해제 클릭 | 확인 단계 `TBD` → 확인 시 고정 영역 비움 |
| 연필 클릭 | 편집 화면·포스트 선택 `TBD` |
| (고정 지정) | 와이어는 이미 고정된 상태만 표시. 지정 플로우 `TBD` |

## 정책 (확인 필요)

- 동시 **고정 개수** 상한(1개만 등): 제품 정책 확인.
- VIP 전용 포스트 고정 시 노출: `mem-M2.md`와 정합.
