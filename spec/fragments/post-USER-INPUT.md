## PM-2 — UI 요소 (부록 기준)

| 요소 | 있음? | 비고 |
|------|-------|------|
| 포스트 제목 목록 | ✓ | 썸네일/제목 컬럼 — **목록 부록 미확보**, 와이어 기준 |
| 상태(임시저장·게시·예약) | ✓ | 상단 탭: 전체·발행됨·예약됨·임시저장(건수 배지). 행별 상태 뱃지(발행됨/예약됨/임시 저장) |
| 검색 | ✓ | 검색 입력 필드 |
| 필터(정렬·멤버십) | ✓ | 최신순 드롭다운, 전체 멤버십 드롭다운 (날짜 범위 필터는 부록에 없음) |
| 신규 작성 CTA | ✓ | **+ 새 포스트 작성** (우측 상단 빨간 버튼) |
| 행 선택·일괄 삭제 | ✓ | 행 체크박스, 하단 **선택한 포스트** → **삭제** |
| 행별 편집 | ✓ | 목록에서 행 진입 시 편집 화면(`_1`). 복제는 부록에 없음 → `TBD` |
| 가시성/멤버십 노출 | ✓ | 권한 열: 전체 공개, 베이직/프리미엄/VIP 구독 등 (홈 `home-H7`과 용어 맞출 것) |

**편집 화면:** 제목·썸네일(JPG/PNG 최대 5MB)·본문 리치 텍스트·우측 **현재 상태**·**공개 설정**·**열람 권한 설정**(구독형 복수·세션형 1개·상호 배제 경고)·**댓글 게시판**·**열람 기간 설정**·**미리보기**·**변경사항 저장** (`_1` 전체, `_2`는 우측 패널 근거).
