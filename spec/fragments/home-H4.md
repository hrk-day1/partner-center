# H-4 — 큐레이션 목록 헤더 · 배지(이중 구조)

<!-- CP-HOME 확정: 2026-03-31 — home-USER-DECISIONS O-02 -->

## UI

- 섹션 제목: **큐레이션 목록**
- 섹션 헤더 배지: **N개** 형식(부록 예: **3개**).
- 우측 CTA: **큐레이션 추가** (`home-H5.md`).

## 부록에서 관측된 이중 숫자

- 섹션 옆 **3개**와, 그룹 행(예: 인기 콘텐츠) 옆 **3**은 **서로 다른 단위**다.

## 의미 (확정)

| 대상 | 정의 (O-02) |
|------|-------------|
| 섹션 헤더 배지 **N개** | **큐레이션 추가**로 생성된 **큐레이션 그룹**의 개수(목록에 올라온 그룹 수). |
| 그룹 행 옆 숫자 | 해당 그룹에 속한 **포스트(행) 개수**. |

## 명세 문장 (확정)

> 「큐레이션 목록」 헤더 옆 배지는 **사용자가 생성한 큐레이션 그룹의 개수**를 나타낸다. 각 그룹 제목 옆 숫자는 **그 그룹 안에 담긴 포스트 행 수**를 나타낸다.
