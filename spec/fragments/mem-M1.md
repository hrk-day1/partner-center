- 본문 **최상단**에 홈·포스트 편집과 동일한 **미리보기 / 변경사항 저장** 한 쌍은 **없음**.
- 테이블 **작업** 열에 **미리보기(눈 아이콘)** — 상품 상세 확인용으로 해석.

## 화면 구조

| 구역 | 내용 |
|------|------|
| 좌측 내비 | Creator Center — **멤버십 상품** 선택(강조) |
| 제목·부제 | 멤버십 상품 — 관리자 등록 상품 현황, 미리보기로 상세 확인 안내 |
| 대시보드 카드 | 제목 「대시보드」— 범위 확정 후 시각화 등 **개발 범위에 따라 구현** 문구(플레이스홀더) |
| 검색·필터 | ID·상품명 검색, **판매 상태**·**상품 유형** 드롭다운(기본 전체) |
| 상품 테이블 | 아래 열 구성 |

## 테이블 열 (부록)

| 열 | 설명 |
|----|------|
| ID | 행 번호 |
| 상품명 | 예: 베이직 구독, 프리미엄 구독, VIP 구독, 베이직 세션, 프리미엄 세션 |
| 상품 유형 | 태그: **구독형** / **세션형** |
| 가격 | 월 단위(원/월) 또는 세션 단가(원) |
| 판매 상태 | **판매중** / **판매중단** / **준비중** 등 뱃지 |
| 판매 기간 | 기간 또는 **상시** |
| 리워드 | 숫자(개) |
| 작업 | 미리보기(아이콘) |

## 부록에 나온 상품 예시 (명칭 참고)

1. 베이직 구독 — 구독형, 4,900원/월, 판매중, 상시, 리워드 3  
2. 프리미엄 구독 — 구독형, 9,900원/월, 판매중, 기간 지정, 리워드 4  
3. VIP 구독 — 구독형, 19,900원/월, 판매중단, 기간 지정, 리워드 5  
4. 베이직 세션 — 세션형, 12,000원, 판매중, 기간 지정, 리워드 2  
5. 프리미엄 세션 — 세션형, 25,000원, 준비중, 상시, 리워드 3
